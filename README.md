# 医院管理的前端内容

[医院管理后端地址](https://github.com/rui-jim/hospital-manage) https://github.com/rui-jim/hospital-manage

## 项目描述：使用Vuex存储信息的状态，并在其他组件中使用。熟练使用ElementUI进行页面的快速搭建使用，axious进行请求的发送以及过滤。以及使用VueRouter进行页面的路由访问
